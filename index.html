Usage

// Initialize the virtualization system
Context context = getApplicationContext();
LoggingSystem logger = new LoggingSystem(context);
ConfigurationManager configManager = new ConfigurationManager(context, logger);
VirtualizationManager virtualizationManager = new VirtualizationManager(context, configManager, logger);
virtualizationManager.initialize(); // This now also initializes GAID/OAID hooks

// Install a virtual application
String apkPath = "/path/to/app.apk";
int userId = 0;
virtualizationManager.installApplication(apkPath, userId);

// Launch a virtual application
String packageName = "com.example.app";
virtualizationManager.launchApplication(packageName, userId);

// Optionally, manage GAID settings for an app
VirtualAdvertisingIdManager adIdManager = RedBoxMain.getInstance().getVirtualAdvertisingIdManager();
if (adIdManager != null) {
    adIdManager.setLimitAdTracking("com.example.app", true);
    String virtualGaid = adIdManager.getVirtualGAID("com.example.app");
    // Use virtualGaid or confirm settings
}

Security Controls

Device ID Spoofing

RedBox provides comprehensive device ID spoofing capabilities to protect user privacy and enable testing across different device profiles.

Network Location Spoofing

Advanced network location spoofing allows you to test location-based features without physical movement.

Container-Level Security

Each virtualized application runs in its own secure container with isolated storage, processes, and permissions.

Penetration Testing

Built-in penetration testing tools help identify and mitigate potential security vulnerabilities in your virtualized environment.

Notification System

Comprehensive notification handling with NotificationManager and NotificationService, including support for notification channels and importance levels.

Cross-App Communication

Secure cross-application communication via the CrossAppCommunicationController, allowing controlled interaction between virtualized apps.

Security Mechanisms

Robust security features including integrity checking and runtime verification to ensure system integrity and prevent tampering.
